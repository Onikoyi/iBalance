# Domain Boundaries - iBalance

## Platform Module
Responsibilities:
- Authentication
- Authorization
- Tenant resolution
- User and role management
- Audit logging
- Notifications
- Approval framework foundation

## Finance Module
Responsibilities:
- Chart of Accounts
- General Ledger
- Journals
- AP
- AR
- Cash and bank
- Budgeting
- Financial period controls
- Reporting foundation

## Universities Module
Responsibilities:
- Student billing accounting integration
- Grants and restricted fund support
- Departmental budget views
- Academic service billing support

## OilAndGas Module
Responsibilities:
- Project and field cost tracking
- Cost allocation support
- Operational expenditure grouping
- Sector reporting support

## Shared Building Blocks
Responsibilities:
- Base abstractions
- Domain events
- Result handling
- Common utilities
- Shared contracts
